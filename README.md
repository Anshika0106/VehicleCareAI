# VehicleCare AI - Predictive Maintenance System

A predictive maintenance system for automotive vehicles using Python and Streamlit.

## Features

- Real-time vehicle telemetry simulation
- Anomaly detection using Isolation Forest
- AI-powered maintenance recommendations
- Interactive dashboard with live charts

## Deployment

This app can be deployed on Streamlit Cloud, Heroku, or any platform that supports Python.

### Streamlit Cloud Deployment

1. Push this repository to GitHub
2. Go to https://share.streamlit.io
3. Sign in with GitHub
4. Click "New app"
5. Select your repository and set the main file to `app.py`
6. Click "Deploy"

## Local Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```
